# Starhauler

Pick Up and Deliver Goods around the Universe

## Play Online

Play the game at: [https://bsproger.github.io/starhauler/](https://bsproger.github.io/starhauler/)

## About

Starhauler is a space logistics idle game where you manage mining facilities and build a fleet of explorer ships in the Saturn ring system.

## Development

This is a static HTML/CSS/JavaScript game with no build process required. Simply open `index.html` in a web browser to play locally.

## Deployment

The game is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.
